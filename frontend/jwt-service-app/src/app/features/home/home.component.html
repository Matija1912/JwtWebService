ABOUT JWS

WHAT IS THIS ABOUT?

PURPOSE

This project was developed as part of a bachelorâ€™s degree program. It provides a JSON Web Token (JWT) service built on a custom Node.js library implemented in C (jwt-native-implementation). The library includes native implementations of HMAC-SHA256 and Base64 URL encoding/decoding, used for token generation, signing, and verification.

FUNCTIONALITY

Registered users get access to an API for handling authentication without having to manage their own user storage. The service stores user data in its own database and provides a user management interface with login and logout history.

GOAL

The goal is to offer a functional, self-contained authentication service that can be integrated into other applications with minimal setup.
